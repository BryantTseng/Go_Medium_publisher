# GO_Medium_Publisher

## Post .md file to Medium

Use [Medium API](https://github.com/Medium/medium-api-docs) to post new article to medium

### Usage

#### build executable

```shell
go build
```

#### Post to Medium

```shell
./Go_Medium_Publisher {MARKDOWN_FILE}
```

#### Example

```shell
./Go_Medium_Publisher examplefiles/example.md
```
